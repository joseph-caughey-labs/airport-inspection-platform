# Airport Inspection Platform

A production-style demonstration of an AI-assisted airport inspection platform: real-time sensor ingestion, computer-vision event detection, 10-layer validation, and an operations dashboard.

## Status

Pre-Phase 1 — repo scaffolding only. Implementation lands via feature branches under the Git Flow process.

## Process

This repo follows [Git Flow](#git-flow): `main` is always demo-runnable, `develop` is the integration branch, all work flows through `feature/*`, `release/*`, and `hotfix/*` branches. See the Git Flow process document (kept in the project planning folder alongside the brief, folder structure, role catalog, and sprint plans).
